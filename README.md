# Armenia Flood Information Platform (FIP)

A citizen-science based interactive map to visualize flood events in Armenia over time.

## How to use
1. Add flood video links to `data/floods.json`.
2. Open `index.html` in your browser.
3. Use the timeline slider at the bottom to filter events by date.

## Technology
- **Leaflet JS**: For the interactive map.
- **noUiSlider**: For the time range selection.
- **Vanilla JS**: No complex frameworks needed.
